# Desafio-Jogo-War
Projeto em C inspirado no jogo WAR, com batalhas por dados, conquista de territórios e sistema de missões dinâmicas. Usa structs, ponteiros, alocação dinâmica, controle de estado e lógica de jogo, com código modularizado e totalmente comentado, focado em boas práticas e lógica de programação.
